# Sacred IDs tool by Ethan94FL

## Per <ins>Sacred: la leggenda dell'arma sacra</ins>

Questo semplicissimo e basilare tool mette a disposizione una cospicua lista di ID riguardanti differenti elementi del gioco:

1. Mobs
2. Oggetti
3. Arti di combattimento/magie
4. Oggetti missione
5. Oggetti scenici
6. Equipaggiamenti vari

## Come usare il tool

1. Avviare il tool aprendo il file index.html (la homepage)
2. Cerca nella barra di ricerca un ID o il nome di ciò che cerchi
3. Clicca sul risultato per copiare l'ID

>NB: è importante che i file `.js` siano tutti presenti nella stessa cartella del tool, accompagnati dalla cartella `src`.
